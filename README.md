# Open Source Audit — Python

Name: Ansh Arora 
Reg No: 24BAI10709 

## Description
This project is an audit of Python as an open-source software. It includes analysis of its origin, license, ecosystem, and comparison with proprietary software. It also contains shell scripts demonstrating basic Linux operations.

## Scripts

script1.sh - Displays system information 
script2.sh - Checks if Python is installed 
script3.sh - Shows Python directory and permissions 
script4.sh - Analyzes system logs 
script5.sh - Takes user input and generates output.txt 

## How to Run

chmod +x script.sh 
./script.sh
